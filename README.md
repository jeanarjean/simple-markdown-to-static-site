simple-markdown-to-static-site
===========================================

Turn a folder and its markdown files into an easy to read static website.

`To install on your machine`
-------
1. npm install
2. npm run build
3. npm install -g

`To install in development mode`
-------
Commands
1. npm install
2. npm run build
3. npm link

`To run`
-------
gen-static-site
